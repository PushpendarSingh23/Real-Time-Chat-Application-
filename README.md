# Real-Time Chat Application

A scalable real-time messaging platform enabling instant communication between users using WebSockets.

## 🚀 Features
- Real-time messaging using Socket.IO
- User authentication system
- Instant message delivery
- Scalable backend communication
- Secure message handling

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- Socket.IO

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/YOUR_USERNAME/Real-Time-Chat-Application.git
```

Navigate to project directory

```
cd Real-Time-Chat-Application
```

Install dependencies

```
npm install
```

Start the server

```
npm start
```
